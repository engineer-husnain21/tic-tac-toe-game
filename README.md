A simple 2-player Tic Tac Toe game built using HTML, CSS, and JavaScript. 
Made this project to practice DOM manipulation and understand how 
JavaScript can be used to handle click events and update the game 
state in real time. 

#Features
- Two players can play by taking turns (X and O)
- Automatically checks for winner after each move
- Shows a message when someone wins or if it's a draw
- Reset/Restart button to play again

#Technologies Used
 HTML
- CSS
- JavaScript
